# Prerequisites

Have Rust SDK/CDK and the Big Map in the same directory:

```bash
git clone git@github.com:dfinity-lab/rust-cdk.git
git clone git@github.com:dfinity-lab/big-map-rs.git
git clone git@github.com:dfinity-lab/sdk.git
```


```bash
```

```bash
rustc --version

```
